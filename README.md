Сколько типов данных существует в JS?

    Существует 6 типов данных:
        1. Число "number",
        2. Строка "string",
        3. Булевый тип "bolean",
        4. Специальное значение "null",
        5. Специальное значение "undefined",
        6. Объекты "object";
